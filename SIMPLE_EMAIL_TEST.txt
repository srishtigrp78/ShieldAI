📋 TESTING STEPS:

1. KILL OLD PROCESSES:
   lsof -ti:8080 | xargs kill -9
   lsof -ti:3000 | xargs kill -9

2. START DASHBOARD:
   cd /Users/srishtigupta/ShieldAI/dashboard
   mvn spring-boot:run

3. TEST API DIRECTLY:

        -H "Content-Type: application/json" \
        -d '{"email":"hr@company.com"}'

   Expected response: {"message":"Test email sent to hr@company.com. Check emails/test_email_2025-09-07_15-30-45.txt"}

5. VERIFY EMAIL FILE:
   cd /Users/srishtigupta/ShieldAI/dashboard
   ls emails/
   cat emails/test_email_*.txt

6. START FRONTEND (if needed):
   cd /Users/srishtigupta/ShieldAI/frontend
   npm start
